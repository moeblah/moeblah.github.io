# Install Docker on Ubuntu 20.04

Ref : https://docs.docker.com/engine/install/ubuntu/open in new window

Install Docker on Ubuntu 20.04

  1. Update the apt package index and install packages to allow apt to use a repository over HTTPS:
sudo apt update
sudo apt install apt-transport-https ca-certificates curl gnupg lsb-release
1
2
  1. Add Docker's official GPG key:
curl -fsSl https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
1
  1. Use the following command to set up the stable repository.
echo \
  "dev [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
sudo apt update
1
2
3
4
  1. install Docker Engine
sudo apt install docker-ce docker-ce-cli containerd.io
1
  1. Verify that Docker Engine is installed correctly by running the hello-world image.
sudo docker run --rm hello-world
1
  1. Add your user to the docker group.
sudo usermod -aG docker $USER
1
  1. Log out and log back in so that your group membership is re-evaluated.
